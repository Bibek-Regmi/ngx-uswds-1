When Date Picker Calendar is open:

Tab / Shift + Tab - Move focus between calender header buttons and calender body Up/Down/Left/Right Arrow - Navigate across each cell of calender body when focus is on calender body Enter / Space - Select a Date Home / End - Move focus to first/last cell on calender body End - Move focus to last cell on calendar body Page Up / Page Down - Go to next month on month view or next set of years on multi year view Escape - Close the calendar

General Accessibility

The calendar grid provides hotkeys for changing the year and month as well as support for normal grid navigation keys.

When the month or year of the calendar grid changes as users navigate the calendar or activate the buttons for next or previous month or year, a live region enables screen readers to announce the new month and year.

The cells of the calendar body are navigable by keys while the over all calender buttons can be navigated using tab keys, which allows users to quickly navigate to their desired date on the calendar

After user selects a date, the calender automatically closes and focus moves to the text input, allowing screen readers to confirm the user's date selection
